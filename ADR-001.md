# ADR-001: Архитектура проекта "Chill" (V3 Edition)

## Статус
Принято (Accepted)

## Контекст
Нам необходимо создать современный, адаптивный лендинг с высокой конверсией, SEO-оптимизацией и интуитивно понятной админ-панелью. Проект должен следовать принципам Mobile-first и использовать современные UI-тренды.

## Решение: Tech Stack
Мы выбираем следующий стек для максимальной производительности и SEO:

- **Frontend**: SvelteKit (лучший для SEO и скорости благодаря SSR/SSG).
- **Styling**: Tailwind CSS + Headless UI (адаптивность и кастомный дизайн).
- **UI Style**: Glassmorphism (эффект матового стекла) + чистые градиенты.
- **Backend/DB**: Supabase (Auth, PostgreSQL, Realtime).
- **SEO**: Semantic HTML5, JSON-LD Schema.org, Meta-tags manager.

## Дизайн-система (UI/UX)
1. **Landing Page**:
   - Hero-секция с видео-фоном или качественным рендером.
   - Sticky-навигация с blur-эффектом.
   - Анимированные карточки услуг (Framer Motion или Svelte Transitions).
   - Форма захвата лидов с валидацией в реальном времени.

2. **Admin Dashboard**:
   - Темная тема по умолчанию (Dev-friendly).
   - Sidebar с иконками Lucide.
   - Дашборд с графиками (Chart.js/LayerChart).
   - Управление контентом через WYSIWYG редактор.

3. **SEO & Performance**:
   - 100/100 Lighthouse score (цель).
   - Автоматическая генерация Sitemap и Robots.txt.
   - Оптимизация изображений (WebP/AVIF).

## Последствия
- Высокая скорость разработки за счет использования готовых компонентов.
- Отличная индексация поисковиками.
- Легкая масштабируемость через Supabase.
